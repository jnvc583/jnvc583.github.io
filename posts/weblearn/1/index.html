入门

今天，我们来学习HTML的基础知识。
首先，你得知道HTML是什么，它由什么组成。
HTML中有很多标签，如<html>、<head>、<body>等。它们还有一定的结束标签，比如</html>、</head>、</body>等。
这些标签有各自的功能，如<b>标签用于加粗文字</b>。

在HTML中，<html>标签是整个HTML文档的根元素，它包含文档的所有内容。<head>标签用于定义文档的头部信息，如标题、元数据等。<body>标签用于定义文档的主体内容，整个HTML文档的所有展示的内容都包含在<body>标签中。

接下来，我们来学习一些常用的HTML标签：

  • <h1>到<h6>：用于定义标题，数字越小，标题级别越高。
  • <p>：用于定义段落。
  • <a>：用于定义超链接，可以通过href属性指定链接地址。
  • <img>：用于插入图片，可以通过src属性指定图片地址。
  • <ul>和<ol>：分别用于定义无序列表和有序列表。
  • <li>：用于定义列表项，必须包含在<ul>或<ol>标签内。
  • <div>：用于定义块级元素，可以用于布局和样式。
  • <span>：用于定义行内元素，可以用于样式。
  • <s>：用于定义删除线文本。
  • <u>：用于定义下划线文本。
  • <nobr>：用于定义不换行文本。

那么<!doctype html>是干什么的呢？

<!doctype html>声明用于告诉浏览器当前文档使用的是HTML5标准，这样浏览器就能正确地解析和渲染页面内容。

接下来看一看一个html实例：

<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

这个实例展示了一个简单的HTML页面结构。它包含了DOCTYPE声明、html标签、head标签和body标签。在head标签中，我们定义了字符编码和视口设置，以及页面标题。在body标签中，我们使用h1标签定义了一个标题，使用p标签定义了一个段落。
